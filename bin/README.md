# bin scripts (maintainer-local)

These scripts are **maintainer devops utilities** used in Kenzo's local workflow.

They are **not part of the public Pi Rig extension API** and should be treated as optional local tooling.

Examples in this folder include:
- plan relay/watch helpers
- Telegram notification helpers
- local markdown index maintenance scripts

If you are using Pi Rig as a consumer, you can ignore this folder.
