# @codewithkenzo/pi-rig

One-command installer for Pi plugins that are available now.

Current install scope:

- `@codewithkenzo/pi-dispatch`
- `@codewithkenzo/pi-theme-switcher`

## Usage

```bash
bunx @codewithkenzo/pi-rig@latest
# or
npx @codewithkenzo/pi-rig@latest
```

Optional flags:

```bash
bunx @codewithkenzo/pi-rig@latest --all
bunx @codewithkenzo/pi-rig@latest --extensions flow-system,theme-switcher
bunx @codewithkenzo/pi-rig@latest --dry-run
bunx @codewithkenzo/pi-rig@latest --pi-path /absolute/path/to/pi
```

More plugins are planned for later phases.
